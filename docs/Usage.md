# 1. Installation and usage







## Installation

Prerequisites: Node, yarn

From this project directory:

````
$ yarn install
$ yarn link
````

## Link the module from your app


From your app folder:

````
$ yarn install
$ yarn link @kalisio/feathers-keycloak-listener
````
